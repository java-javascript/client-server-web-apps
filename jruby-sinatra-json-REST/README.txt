To build the project:

	> mvn clean install

To start the server

	> mvn test -Pserver
	
Hit the main page at

	http://localhost:4579/
	
This demonstrates
	- ruby is available
	- rubygems (sinatra) available
	- java (System) available
	
Tested on Mac OSX with:
	Apache Maven 2.2.1 (r801777; 2009-08-06 15:16:01-0400)
	Java version: 1.5.0_19
